## 📖 Descripción

Aplicación desarrollada para la administración y control de equipamiento informático dentro de una organización. El sistema permite gestionar inventario tecnológico, registrar dispositivos y facilitar el seguimiento del estado y asignación de recursos.

El proyecto está orientado a mejorar la organización y mantenimiento de activos tecnológicos de manera centralizada.

---

## 🚀 Características

- Registro de dispositivos informáticos
- Gestión de inventario
- Control de asignaciones
- Seguimiento del estado del equipamiento
- Búsqueda y filtrado de recursos
- Interfaz responsive
- Gestión de roles

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- PHP
- MySQL

---

## 🎯 Objetivos del proyecto
- Centralizar la gestión de equipamiento.
- Mejorar el control de inventario.
- Facilitar el seguimiento de recursos tecnológicos.
- Optimizar tareas administrativas.
